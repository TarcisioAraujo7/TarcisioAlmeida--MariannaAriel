# Jogo Procurando Nemo.

## Descrição.
O jogo segue a historia do filme Procurando Nemo(2003), onde o seu objetivo é avançar no cenario até encontrar o Nemo.
- Cuidado com salas perigosas.
- Leia os dialogos com atenção.
